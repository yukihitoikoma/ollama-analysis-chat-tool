# CSVデータ分析チャットアプリ

このアプリケーションは、Streamlitフレームワークを使用して構築されたCSVデータ分析ツールです。ユーザーはCSVファイルをアップロードし、チャットインターフェースを通じてデータに関する質問を行うことができます。

## 機能

- CSVファイルのアップロードと読み込み
- データの基本統計情報表示
- データプレビューと欠損値確認
- チャットインターフェースでのデータ分析
- データの可視化機能（今後の実装予定）

## セットアップ手順

1. このリポジトリをクローンまたはダウンロードします
2. 必要な依存関係をインストールします:

```bash
pip install -r requirements.txt
```

3. アプリケーションを実行します:

```bash
streamlit run src/app.py
```

## 使用方法

1. アプリケーションを起動します
2. 「CSVファイルをアップロードしてください」ボタンからCSVファイルを選択します
3. ファイルが正常に読み込まれると、以下の情報が表示されます:
   - ファイル情報（名前、サイズ、行数、列数）
   - 列名とデータ型
   - データプレビュー（最初の10行）
   - 基本統計情報
   - 欠損値の確認
4. データが正常に読み込まれた後、チャットインターフェースでデータに関する質問を入力できます
